## Git Commands to View Branches
```
git branch -a          # List all branches
git status             # Current branch
git log --oneline      # Commits (branch shows in VSCode Git tab)
```

## Fix "Not Showing" in VSCode
1. Open **Source Control** (Ctrl+Shift+G).
2. Click refresh icon.
3. If remote, run `git fetch`.
